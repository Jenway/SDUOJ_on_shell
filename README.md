# SDUOJ_on_shell

## 这是什么？

> This repo is still under construction.

这是一个 python 项目，在命令行上使用 [SDUOJ](https://github.com/SDUOJ/OnlineJudge)

## 可以做什么？

- [X] 在终端登录 SDUOJ
- [X] 查看题目列表与题目详情
- [] 提交代码
- [] 查看提交状态

## How does it work?

1. 通过山大 SSO 登录 SDUOJ 并获取 cookie
2. 调用 SDUOJ 的 API
